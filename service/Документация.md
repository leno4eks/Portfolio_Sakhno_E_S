## 1. Постановка задачи

**Бизнес-задача:**  
Внедрить блок «Рекомендуемые товары» на главной странице для увеличения среднего чека на 20% через кросс-сейлз.

**Техническая метрика:**  
**Precision@3** — доля купленных товаров среди топ-3 рекомендаций.

## 2. Описание данных

- `events.csv` — события пользователей (2.76 млн записей)
- `item_properties_part1/2.csv` — свойства товаров (20.3 млн записей)
- `category_tree.csv` — дерево категорий (1669 категорий)

## 3. Предобработка и Feature Engineering

- Преобразование timestamp в datetime
- Генерация признаков:
  - **Пользовательские**: активность, конверсия, количество сессий
  - **Товарные**: популярность, конверсия, свежесть, уровень категории
  - **Взаимодействия**: история user-item, интенсивность
- Временной сплит: train — до 01.08.2015, test — после

## 4. Моделирование

**Выбранная модель:** XGBoost Ranker (rank: pairwise)

**Гиперпараметры:**
- n_estimators=500
- max_depth=6
- learning_rate=0.06
- subsample=0.8, colsample_bytree=0.8

**Результат:**  
**Precision@3 = 87.35%**

## 5. Веб-сервис

- Фреймворк: **FastAPI**
- Основной эндпоинт: `GET /recommend/{visitor_id}`
- Возвращает топ-3 товара с score
- Swagger UI доступен по адресу `/docs`

## 6. Контейнеризация (Docker)

**Dockerfile:**

```dockerfile
FROM python:3.10-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
``` 

### Команды:

```
Bashdocker build -t recommendation-service .
docker run -p 8000:8000 recommendation-service
```
