## Описание проекта

Этот проект направлен на выявление недобросовестных отелей, которые могут завышать свои рейтинги. Основная задача - построить модель машинного обучения, которая предсказывает рейтинг отеля на основе различных признаков. Если прогнозы модели значительно отличаются от фактического рейтинга, это может указывать на возможные манипуляции с оценками.

## Структура проекта

### Блок 1: Загрузка данных
- Импорт необходимых библиотек
- Загрузка тренировочных и тестовых данных
- Объединение датасетов для корректной обработки признаков

### Блок 2: Подготовительный этап
- Настройка статистических функций для анализа
- Определение уровня значимости (alpha = 0.05)
- Создание вспомогательных функций для визуализации и статистических тестов

### Блок 3: Знакомство с данными
- Исследование структуры данных
- Анализ пропущенных значений
- Изучение распределения признаков

### Блок 4: Работа с признаками

#### Географические данные
- Восстановление пропущенных координат (lat, lng)
- Извлечение города и страны из адреса отеля
- Расчет расстояния до центра города
- Создание признака "хорошая локация"

#### Временные признаки
- Извлечение месяца, года, недели и квартала из даты отзыва
- Создание признака "первые три месяца года"

#### Анализ отзывов
- Обработка текстовых отзывов (положительных и отрицательных)
- Подсчет количества слов в отзывах
- Анализ тональности с использованием VADER
- Создание признаков полярности и категоризации отзывов

#### Работа с тегами
- Извлечение информации о типе комнаты
- Определение типа поездки (соло, пара, семья, группа)
- Анализ продолжительности пребывания
- Определение способа бронирования (мобильное устройство)

#### Дополнительные признаки
- Ранжирование отелей по популярности
- Создание рейтингов по почтовым индексам
- Анализ национальности рецензентов и их локации

## Ключевые особенности

### Статистический подход
- Проверка нормальности распределений (тест Шапиро-Уилка)
- Использование непараметрических тестов (Манна-Уитни, Крускала-Уоллиса)
- Расчет эффекта с помощью коэффициента Клайна

### Feature Engineering
- Создание 50+ новых признаков
- Комбинирование географических, временных и текстовых данных
- Ранжирование и агрегация данных на разных уровнях

### Визуализация
- Интерактивные графики распределения
- Тепловые карты корреляций
- Круговые диаграммы и барплоты

## Технические детали

### Используемые библиотеки
- **Анализ данных**: pandas, numpy
- **Визуализация**: matplotlib, seaborn
- **Машинное обучение**: scikit-learn
- **Статистика**: scipy, statsmodels
- **Обработка текста**: nltk, textblob
- **Геоданные**: geopy

### Метрики оценки
- Основная метрика: MAPE (Mean Absolute Percentage Error)
- Дополнительные метрики: MAE, RMSE

Проект демонстрирует комплексный подход к:
