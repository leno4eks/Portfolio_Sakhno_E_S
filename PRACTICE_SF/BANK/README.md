## Описание проекта

Этот проект посвящён решению задачи бинарной классификации для прогнозирования оттока клиентов банка. В рамках работы построены и проанализированы модели машинного обучения, включая логистическую регрессию, дерево решений и случайный лес, с целью определения вероятности ухода клиента и оценки качества моделей.

[к оглавлению](#оглавление)

## Структура проекта 

- **Исследование данных (EDA)**: Анализ распределения целевой переменной, взаимосвязей между признаками, визуализация данных.
- **Подготовка данных**: Обработка пропусков, кодирование категориальных признаков, масштабирование числовых признаков.
- **Моделирование**:
  - Логистическая регрессия
  - Дерево решений
  - Случайный лес
- **Оценка моделей**: Использование метрик (точность, полнота, F1-score, ROC-AUC) и анализ важности признаков.
- **Интерпретация результатов**: Выводы по эффективности моделей и рекомендации для бизнеса.

## Данные

Файл `churn.csv` содержит информацию о клиентах банка:
- **Целевая переменная**: `Exited` (1 — клиент ушёл, 0 — остался)
- **Признаки**: демографические данные, финансовые показатели, активность клиента и др.

## Основные шаги выполнения

1. **Импорт библиотек и загрузка данных**
2. **Разведывательный анализ (EDA)**:
   - Распределение целевой переменной
   - Анализ баланса клиентов
   - Взаимосвязи между признаками
3. **Предобработка данных**:
   - Обработка категориальных признаков
   - Масштабирование числовых признаков
   - Разделение на обучающую и тестовую выборки
4. **Построение моделей**:
   - Логистическая регрессия
   - Дерево решений
   - Случайный лес
5. **Оценка и сравнение моделей**:
   - Метрики классификации
   - Важность признаков
6. **Интерпретация результатов**

## Результаты

- Лучшая модель: **Случайный лес** с показателем  ~0.69
- Ключевые признаки, влияющие на отток: `Age`, `Balance`, `NumOfProducts`
- Рекомендации для банка: внедрить систему раннего предупреждения оттока на основе модели
