# Анализ A/B-тестирования посадочных страниц турагентства

## Описание проекта

Турагентство протестировало два варианта посадочной страницы (А – контроль, В – тест) для увеличения продаж туров.  
Цель: определить, какой вариант эффективнее по двум ключевым метрикам – конверсии в покупку и ежедневному среднему чеку.

## Данные

Файл: `ab_data_tourist.csv`  
Поля:
- `user_id` – идентификатор пользователя
- `date` – дата посещения
- `group` – группа теста (A или B)
- `purchase` – совершена ли покупка (1 – да, 0 – нет)
- `price` – цена купленного тура (0, если покупки не было)

Предлагаемые туры:
- Таиланд – 100 000 руб.
- Турция – 60 000 руб.
- Мальдивы – 200 000 руб.
- Санкт-Петербург – 10 000 руб.
- Камчатка – 150 000 руб.

## Этапы анализа

1. **Первичная обработка данных**  
   - Преобразование типов, проверка пропусков.
   - Унификация длительности теста (обе группы собирались одинаковый период).
   - Удаление пользователей, попавших в обе группы (1895 пересечений).

2. **Расчёт ключевых показателей**  
   - Конверсия = покупки / посещения.
   - Средний чек = сумма покупок / количество покупок.
   - Анализ продаж по каждому туру.

3. **Визуализация**  
   - Сравнение конверсии и среднего чека по дням и группам.
   - Боксплоты и линейные графики.
   - Распределение продаж по турам (количество и выручка).

4. **Проверка стабилизации метрик**  
   - Построены кумулятивные графики конверсии и среднего чека.
   - Метрики стабилизировались после 10–12 января → тест можно считать завершённым.

5. **Статистический анализ**  
   - Проверка нормальности (критерий Шапиро–Уилка) – распределения нормальные.
   - Сравнение конверсий: **z-тест для пропорций**.
   - Сравнение среднего чека: **t-тест** (неравные дисперсии).
   - Построены 95% доверительные интервалы для метрик и их разностей.

## Результаты

| Метрика | Группа A | Группа B | p-value | Статистическая значимость |
|---------|----------|----------|---------|----------------------------|
| Конверсия | 12.03% | 11.89% | 0.239 | Нет (p > 0.05) |
| Средний чек | 80 969 руб. | 88 576 руб. | ≈ 0.000002 | Да (p < 0.05) |

**Дополнительные наблюдения:**  
- По продажам туров: группа А лидирует по Турции, группа В – по Таиланду.  
- Группа В генерирует на 115 млн руб. больше выручки (≈ +8%).

## Вывод

При одинаковой конверсии **группа B** демонстрирует статистически значимо более высокий средний чек (+9.4%, +7 600 руб.), что приводит к росту общей выручки.  
Следовательно, **вариант B является предпочтительным** для турагентства.
